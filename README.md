# startpage について

**アクセス** https://fukushima-c.github.io/startpage/

2019/7/22 開始  

* このページは、PWAのテストのために仮のツールとして作成した。  
* Javascript により液晶の対角インチと縦横ピクセルから、縦横の実寸ミリを表示する。  
* ブラウザで表示して「ホーム画面に追加」を行うと、単独アプリとして作動する。
